Viewing Data

The AIS Data Archiver displays the following data:

  • ship name
  • MMSI (Maritime Mobile Service Identity)
  • flag
  • latitude
  • longitude
  • speed
  • heading
  • course
  • destination
  • length
  • width
  • draught
  • call sign
  • IMO (International Maritime Organization) number
  • last report date
  • last report time

Location Parameter

  • North: latitude > 0
  • South: latitude < 0
  • East: longitude > 0
  • West: longitude < 0
  • NorthEast: latitude and longitude > 0
  • NorthWest: latitude > 0 and longitude < 0
  • SouthEast: latitude < 0 and longitude > 0
  • SouthWest: latitude and longitude < 0
