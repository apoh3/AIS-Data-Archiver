Viewing Data

With the AIS Data Archiver, you can:

  • view data for a specific MMSI number
  • view a sample of data associated with selected parameters on the Locate Page
  • visually view data on a world map, including historical trackings, on the Locate Page
  • view all data associated with selected parameters on the Edit Page

How to Select Parameters for Viewing

  1. To search for a ship associated with a specific MMSI number, type the number in the text field that says "enter mmsi". Then click the Search Button. (This can be done in the Locate and Edit Pages.)
  2. To search for ships associated with certain parameters, click the arrow of the dropdown box for the parameter(s) you wish to search for. (For "last report date", click the grey box to the right of the text field. Select a date on the calendar, or type a date in the format mm/dd/yyyy.) Once all parameters are selected, click the Search Button.

How to Visually View Data

  1. After clicking the Search Button, double click on a ship in the table of data (bottom left corner of the Locate Page).
  2. The table will display the trackings of the selected ship, and the map representation will update to pin the location(s) of the ship.
